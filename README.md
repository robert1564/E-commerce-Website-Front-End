# E-commerce-Website-Front-End
Front-end part for an e-commerce website. 

Technologies I used to create this project:

  - Html;
  - Css;
  - JavaScript;
  - JQuery;
  - Boostrap;

Best website features:

  - Responsivnes;
  - Sticky Navigation;
  - Smooth Scrolling;
  - Burger Menu;

Case of use:

  - Different kind of e-commerce shops;
